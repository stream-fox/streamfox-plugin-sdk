# Media Plugin SDK Node

TypeScript-first Node SDK for StreamHub-compatible media plugins.

## Packages

- `@streamhub/media-plugin-sdk`: core models, plugin builder, validators, protocol errors.
- `@streamhub/media-plugin-http`: Hono server adapter + built-in frontend.
- `create-media-plugin`: CLI scaffold command for JS/TS plugin projects.

## Quickstart

```bash
npm install
npm run build
npm test
```
